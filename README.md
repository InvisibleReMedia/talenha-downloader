# Using Talenha Downloader

Once you have created and opened a Workspace and defined a folder, you can download your Talenha repository.

## First Command : Get Talenha Repository

This command gets all files from the Talenha repository.

## Second Command : Read a Talenha File

This command invites you to give a path and a name to get the path file name from the Talenha repository (files).
